# F21 — Pop-out Window

> **Status: Removed.** This feature has been removed. Chrome's side panel API does not support programmatic close, which makes it impossible to close the side panel when a popup opens. Because of this fundamental constraint, a reliable pop-out / separate window experience cannot be built. The pop-out button no longer appears in the header.

There are no behavior specifications, acceptance criteria, or implementation notes for this feature.
